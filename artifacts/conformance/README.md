# Conformance Artifact Layout

Generated by:
- `scripts/run_conformance_with_durability.sh`

## Paths

- `artifacts/conformance/latest/smoke_report.json`:
  machine-readable conformance summary report.
- `artifacts/conformance/latest/*.report.json`:
  per-fixture conformance reports.
- `artifacts/conformance/latest/structured_logs.jsonl`:
  canonical line-delimited telemetry logs with replay-critical metadata.
- `artifacts/conformance/latest/structured_logs.json`:
  pretty-printed structured telemetry logs.
- `artifacts/conformance/latest/structured_log_emitter_normalization_report.json`:
  canonical emitter-normalization audit report (field-lock and validation summary).
- `artifacts/conformance/latest/telemetry_dependent_unblock_matrix_v1.json`:
  downstream bead-unblock matrix mapping required artifacts/fields/evidence.
- `artifacts/conformance/latest/logging_final_gate_report_v1.json`:
  final LOGC-F gate status report over telemetry schema + durability evidence.
- `artifacts/conformance/latest/logging_release_checklist_v1.md`:
  downstream release checklist generated from final gate checks.
- `artifacts/conformance/latest/reliability_budget_report_v1.json`:
  reliability budget gate output containing budget statuses, flake summary, and CI failure envelopes.
- `artifacts/conformance/latest/flake_quarantine_v1.json`:
  rolling flake policy/quarantine artifact with per-test quarantine actions.
- `artifacts/conformance/latest/reliability_budget_gate_validation_v1.json`:
  fail-closed validator report for reliability budget and quarantine artifacts.
- `artifacts/conformance/latest/smoke_report.raptorq.json`:
  RaptorQ sidecar envelope for the smoke report.
- `artifacts/conformance/latest/structured_logs.raptorq.json`:
  RaptorQ sidecar envelope for structured telemetry logs.
- `artifacts/conformance/latest/structured_log_emitter_normalization_report.raptorq.json`:
  RaptorQ sidecar envelope for emitter-normalization telemetry report.
- `artifacts/conformance/latest/telemetry_dependent_unblock_matrix_v1.raptorq.json`:
  RaptorQ sidecar envelope for dependent-unblock matrix artifact.
- `artifacts/conformance/latest/smoke_report.recovered.json`:
  decode-drill recovered payload output.
- `artifacts/conformance/latest/structured_logs.recovered.json`:
  decode-drill recovered telemetry output.
- `artifacts/conformance/latest/structured_log_emitter_normalization_report.recovered.json`:
  decode-drill recovered emitter-normalization report.
- `artifacts/conformance/latest/telemetry_dependent_unblock_matrix_v1.recovered.json`:
  decode-drill recovered dependent-unblock matrix artifact.
- `artifacts/conformance/schema/v1/structured_test_log_schema_v1.json`:

- `artifacts/conformance/schema/v1/reliability_budget_gate_schema_v1.json`:
  schema lock describing required keys and status envelopes for reliability budgets.
- `artifacts/conformance/v1/reliability_budget_gate_v1.json`:
  locked reliability budget spec (REL-BUD-001..006), packet mappings, and flake policy thresholds.
- `crates/fnx-conformance/tests/fixtures/structured_log_*.json`:
  negative validator fixtures used to prove fail-closed behavior for unsupported schema versions and missing replay-critical fields.
- `artifacts/conformance/oracle_capture/legacy_networkx_capture.json`:
  oracle fixture capture metadata from legacy NetworkX.

## Naming conventions

- `<artifact>.raptorq.json`:
  sidecar envelope for `<artifact>`.
- `<artifact>.recovered.<ext>`:
  decode-drill output reconstructed from sidecar packets.
- `<fixture>.report.json`:
  fixture-level conformance comparison report.
