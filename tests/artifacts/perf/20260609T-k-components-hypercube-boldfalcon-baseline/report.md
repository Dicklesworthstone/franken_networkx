## One Lever

Add `_hypercube_k_components`, a closed-form certificate for tuple-label
hypercubes generated by NetworkX/FNX.

The fast path is only used for exact simple `Graph` objects with no self loops
and `flow_func is None`. The certificate requires:

- node labels are all binary tuples of the same dimension `d >= 3`
- exact node set is all `2^d` binary tuples
- exact edge count is `d * 2^(d - 1)`
- every node has degree `d`
- every node's neighbor set is exactly its one-bit flips

Those constraints prove the graph is the `d`-dimensional hypercube. Non-label,
non-certificate, and custom-flow cases still delegate.
